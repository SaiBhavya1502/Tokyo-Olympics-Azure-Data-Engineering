# Tokyo Olympics Azure Data Engineering Project

## Project Overview
This project demonstrates an end-to-end Azure Data Engineering solution using the Tokyo Olympics dataset. The project focuses on data ingestion, storage, transformation, ETL pipeline development, and analytics using Microsoft Azure cloud services.

## Technologies Used
- Azure Data Factory
- Azure Data Lake Storage (ADLS Gen2)
- Azure Databricks
- Azure Synapse Analytics
- PySpark
- SQL

## Project Workflow
1. Ingested the Tokyo Olympics dataset using Azure Data Factory.
2. Stored raw data in Azure Data Lake Storage (ADLS Gen2).
3. Performed data cleaning and transformation using PySpark in Azure Databricks.
4. Built ETL pipelines to process and organize the data.
5. Queried and analyzed the transformed data using Azure Synapse Analytics and SQL.

Successfully developed an end-to-end Azure Data Engineering pipeline using Azure services to process the Tokyo Olympics dataset. Gained hands-on experience in cloud-based data ingestion, transformation, ETL, and analytics.
